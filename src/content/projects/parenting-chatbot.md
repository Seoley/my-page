## 프로젝트 소개

0~36개월 영유아 부모 대상 RAG 기반 육아상담 챗봇 서비스 및 상담내역 관리 플랫폼 개발. 기존 운영 중이던 고객지원 앱과 연동하여 통합 서비스를 제공합니다.

## 담당 업무

- **플랫폼 개발 총괄**: 기존 프로젝트 인수인계 및 개선, 수요기업/앱개발사 협업 조율
- **RAG 시스템 설계**: Sentence-BERT + ChatGPT 기반 검색 증강 생성 아키텍처 설계
- **학습 데이터 구축**: 전문가 협업 기반 Q&A 데이터 제작 및 합성 데이터 생성
- **AI 모델 서빙**: Django API 기반 AI 서비스 배포
- **품질 인증**: 와이즈스톤 AI 시험인증 대응 및 통과

## 핵심 구현

- **RAG 시스템:** Sentence-BERT 임베딩 + Cosine Similarity 유사도 측정 + ChatGPT 응답 생성
- **프롬프트 설계:** Reference 기반 응답, 참조 불가 시 짧은 문장으로 대답하도록 설계
- **API 배포:** Django REST Framework 기반 AI 서비스 배포

## 성과

- 국내 AI 시험인증기관 와이즈스톤 시험성적서 **기술정확도 90.9%** 달성
- 자체 RAG 챗봇 솔루션 개발 및 플랫폼 연동 완료

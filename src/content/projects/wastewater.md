## 프로젝트 소개

A2O 하수처리장 공정을 AI로 예측하고 최적 운전값을 도출하는 시스템입니다. 8단계 순차 예측 파이프라인과 이상탐지, 제어인자 최적화 기능을 제공합니다.

## 담당 업무

- **AI 모델 개발**: 8단계 순차 예측 파이프라인 설계 및 모델 학습
- **API 설계**: 27개 FastAPI 엔드포인트 명세 및 구현
- **이상탐지 시스템**: 공정 데이터 실시간 모니터링 알고리즘 개발
- **최적화 알고리즘**: 송풍량/탄소원/응집제 제어인자 최적값 도출 로직 구현

## 바이브코딩 활용

**1. 도메인 분석**
- 하수처리(A2O 공정) 도메인 지식 제로에서 시작
- Claude/GPT로 기술 문서 분석 → API 명세서 27개 작성

**2. 개발 자동화**
- Claude Code SKILL 생성하여 반복 작업 자동화
- PR 메시지 생성, 이슈 템플릿 작성, 코드 리뷰 체크리스트

**3. 체계적 협업**
- Issue/PR 기반 트래킹 전체 관리
- 바이브코딩 결과물의 품질을 프로세스로 담보

## 핵심 구현

- **8단계 예측 파이프라인:** 유입수 → 방류수 순차 예측
- **이상탐지:** 공정 데이터 모니터링 및 알림
- **제어인자 최적화:** 송풍량/탄소원/응집제 최적값 도출

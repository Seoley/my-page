## 프로젝트 소개

치매 위험 노인 및 인지장애 환자를 대상으로 치매 진단 콘텐츠를 제공하고, 약 복용시간 알림, 가족 연락 등의 보조 서비스를 지원하는 대화형 AI 로봇입니다. 회사 최초로 임베디드 소프트웨어 개발 프로세스를 수립하고, Jetson Nano 기반 실시간 음성 상호작용을 구현했습니다.

## 담당 업무

- **PM 및 개발 총괄**: 사업 실무(R&D), 신규 임베디드 개발 프로세스 수립
- **AI 모델 개발**: 정오답 판단 모델(RoBERTa), 감정분석 멀티모달 모델
- **현장 데이터 수집**: 광주 B 노인복지센터 MOU 체결 및 실증 필드 구축
- **AI 기술 검증**: CERAD-K 기반 치매 진단 콘텐츠 설계 및 시험인증 수행

## 핵심 구현

- **정오답 판단 AI:** 사용자 답변을 정답/유사정답/오답 3종으로 분류 (RoBERTa 기반)
- **감정분석 모델:** STT + 목소리 톤 분석 멀티모달 AI (MFCC 특징 추출)
- **실시간 음성대화:** 음성인식 + ChatGPT 연동 상호작용
- **Docker 패키징:** Jetson Nano 리소스 고려한 컨테이너별 환경 분리

## 기술적 결정

**1. 경량화 모델 선택**
- Jetson Nano 환경에서 LLM 대신 RoBERTa/Sentence-BERT 활용
- 제한된 리소스에서 실시간 추론 가능하도록 최적화

**2. Django API 분리**
- GPU 서버에 AI 모델 서빙, 로봇은 API 호출로 경량화
- 모델 업데이트 시 로봇 소프트웨어 재배포 불필요

**3. MLflow 형상관리**
- 모델 버전 관리 및 실험 추적
- A/B 테스트 및 롤백 용이

사용자 음성 입력을 Jetson Nano에서 처리하여 실시간 대화를 수행합니다. 경량 NLP 모델(정오답 판단, 의도 분류)은 로봇 내부에서 실행하고, 무거운 AI 모델(치매 정도 분석, 감정 분석)은 사내 GPU 서버에서 Django REST API로 서빙합니다.

## 성과

- 광주테크노파크 시험인증 **정확도 90.1%** 달성 및 시험성적서 발급
- 사용자 반응속도 성능 개선 **(5초 → 0.5초, 10x 향상)**
- 광주 B 노인복지센터 MOU 체결
- Jetson Nano 기반 임베디드 소프트웨어 배포
- CERAD-K 기반 인지재활 콘텐츠 웹플랫폼 연동

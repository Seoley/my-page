RAG 기반 챗봇 솔루션

"LLM 서비스의 핵심은 검색 품질과 사용자 경험이다"

기간 2026.01 - 2026.03
역할 PM / AI 개발
팀 구성 4명 (PM, Backend, Frontend, AI)
성과 관공서 대상 실서비스 배포

프로젝트 소개

기업 환경에 맞춤화된 RAG 기반 챗봇 솔루션입니다. CPU/GPU/클라우드 환경별 최적화된 RAG 엔진을 제공하며, 실제 관공서 환경(폐쇄망)에 배포하여 운영 중입니다.

LLM 개발 철학 적용

  • RAG 아키텍처 설계: BM25 + Vector 하이브리드 검색으로 정확도와 속도 균형
  • LLM 성능 비교 테스트: Claude, GPT, 로컬 LLM(Llama, Mistral) 벤치마크 수행
  • 프롬프트 = UX: 사용자 의도 분류기 설계로 적절한 응답 경로 라우팅

핵심 구현

  • 하이브리드 검색: BM25 + FAISS 결합, 한국어 특화(Kiwipiepy 형태소 분석)
  • 의도 분류: 프로그램 추천 / 정보 조회 / 가이드 / FAQ 라우팅
  • 환경 맞춤 배포: CPU(BM25+Embedding), GPU(로컬 LLM), Cloud(API) 3모드

하수처리 공정 지능화 시스템

"바이브코딩으로 생산성을 높이고, 체계적인 프로세스로 품질을 담보한다"

기간 2026.02 - 진행중
역할 PM / AI 개발
규모 27개 API 엔드포인트

프로젝트 소개

A2O 하수처리장 공정을 AI로 예측하고 최적 운전값을 도출하는 시스템입니다. 8단계 순차 예측 파이프라인과 이상탐지, 제어인자 최적화 기능을 제공합니다.

바이브코딩 활용

  • 도메인 분석: 하수처리(A2O 공정) 도메인 지식 제로에서 시작, Claude/GPT로 기술 문서 분석 → API 명세서 27개 작성
  • 개발 자동화: Claude Code SKILL 생성하여 PR 메시지 생성, 이슈 템플릿 작성, 코드 리뷰 체크리스트 자동화
  • 체계적 협업: Issue/PR 기반 트래킹 전체 관리, 바이브코딩 결과물의 품질을 프로세스로 담보

핵심 구현

  • 8단계 예측 파이프라인: 유입수 → 방류수 순차 예측
  • 이상탐지: 공정 데이터 모니터링 및 알림
  • 제어인자 최적화: 송풍량/탄소원/응집제 최적값 도출
